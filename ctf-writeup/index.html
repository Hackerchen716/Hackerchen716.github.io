学习路线建议

如果你是刚入门 CTF 的同学，建议按以下顺序学习：

Web 方向：HTTP 协议 → 信息泄露 → SQL 注入 → 文件上传 → SSRF → 反序列化

Pwn 方向：ret2text → ret2shellcode → ret2libc → ROP → 堆利用

工欲善其事，必先利其器。推荐工具：Burp Suite（Web）、pwntools + IDA Pro（Pwn）、nmap + gobuster（信息收集）。
